Manage API Auth keys

{{ _('webinterface_api_auth', 'API Auth') }}

API Auth keys are used as alternative to username & password and are used by scripts, tools, and applications to authenticate to the gateway. These API Auth keys can only be used to access this gateway, however, any gateway can receive and process commands for any device within it's cluster.

For details on usage, see Gateway API Overview @ yombo.net

Some modules automatically create API Auth keys, don't use these for your own scripts as they may change or go away. Be sure to create your own API Auth keys for your scripts or other applications. It's recommended to periodically 'Rotate' your keys for security.

Be sure to keep these keys safe and secure. These keys can grant full access to the gateway and any devices connected.
