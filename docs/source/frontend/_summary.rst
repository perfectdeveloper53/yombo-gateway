=======================================
Frontend (yombo.frontend)
=======================================

The frontend application is used to manage the gateway and it's devices. It
is also the primary webinterface to monitoring and controlling devices connected
to the gateway and within it's cluster.

Language
===========

The Frontend application is written in Javascript using the Vue + Nuxt frameworks.
