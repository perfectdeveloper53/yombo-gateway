Debug and inspection tools

Developer Tools

View various information, not normally used by users. These tools can help you dig into the gateway system and view lots of extra details about what's going on under the hood.

  • Auth Platforms - All the authentication platform data.
  • Cache - Caches in use
  • Commands - Commands loaded
  • Device Types - Device types loaded.
  • Hooks - View what hooks have been called and how many times.
    • Library hooks called.
    • Module hooks called.
  • Modules - Inspect various module attributes.
  • Nodes - System nodes.
  • Statistics - Various information abou the statistics library.
    • Storage lifetimes
  • Requirements - View Python3 required modules.
  • SSL Certificates - Encryption certificates used for TLS (SSL).
